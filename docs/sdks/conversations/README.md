# Conversations

## Overview

Operations related to conversations.           A conversation is a record of messages between a user and an agent, and is composed of           one or more sessions.

### Available Operations

* [list](#list) - Conversations List

## list

Conversations List

### Example Usage

<!-- UsageSnippet language="python" operationID="conversations_list" method="get" path="/api/v1/conversations/" -->
```python
import os
from syllable_sdk import SyllableSDK, models


with SyllableSDK(
    api_key_header=os.getenv("SYLLABLESDK_API_KEY_HEADER", ""),
) as ss_client:

    res = ss_client.conversations.list(page=0, limit=25, search_fields=[
        models.ConversationProperties.AGENT_NAME,
    ], search_field_values=[
        "Some Object Name",
    ], start_datetime="2023-01-01T00:00:00Z", end_datetime="2024-01-01T00:00:00Z")

    # Handle response
    print(res)

```
