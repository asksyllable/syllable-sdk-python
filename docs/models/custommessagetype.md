# CustomMessageType

Type of custom message. Greeting is for voice; email_template is for email (subject + body).

## Example Usage

```python
from syllable_sdk.models import CustomMessageType

value = CustomMessageType.GREETING
```


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `GREETING`       | greeting         |
| `EMAIL_TEMPLATE` | email_template   |
