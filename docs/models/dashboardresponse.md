| `name`                                          | *str*                                           | :heavy_check_mark:                              | Name of the dashboard                           |

| `rank`                                          | *int*                                           | :heavy_check_mark:                              | Dashboard importance, 0 is the highest          |
| `label`                                         | *str*                                           | :heavy_check_mark:                              | Dashboard label.  Typically report or dashboard |
